### 📆 개발 기간
2021.03 ~ 2021.06

### 🖥️ 개발 인원
5명

### 🔨 개발툴
Netbeans IDE

### 💡 개발 목표
Java를 기반으로 프로그램에 접속하는 클라이언트 간 1:N 다중으로 음성을 통해 양방향 소통을 할 수 있는 메신저 프로그램을 개발함으로써
프로젝트 경험을 쌓고자 시작하게 되었습니다.
